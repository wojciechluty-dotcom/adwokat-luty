Komendy do przebudowy projektu w nowym folderze:

cd ~/Downloads
cp -r "ADW LUTY" "ADW-LUTY-NEW"
cd "ADW-LUTY-NEW"
rm -rf node_modules dist
npm install
npm run build

---

Jeśli nadal będzie problem, spróbuj także wyczyścić cache npm:

cd ~/Downloads/ADW-LUTY-NEW
rm -rf node_modules dist package-lock.json
npm cache clean --force
npm install
npm run build

---

Alternatywnie, jeśli chcesz zostać w oryginalnym folderze:

cd ~/Downloads/"ADW LUTY"
rm -rf node_modules dist .vite
npm cache clean --force
npm install
npm run build
